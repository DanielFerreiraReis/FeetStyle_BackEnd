# FeetStyle_BackEnd
Projeto criado para atender a demanda da disciplina programação web. Este projeto tem como intuito arquitetar o backend do projeto "FeetStyle" já desenvolvido.

# Start project
1. Instale o Composer.
2. Abra o terminal na pasta do projeto.
3. Rode: composer install (instala dependências).
    Rode: composer dump-autoload (opcional, gera autoload).
4. Copie seu .env para o projeto (converse com seu administrador)
    composer require vlucas/phpdotenv
5. (Opcional) Rode scripts do Composer: composer run nome-do-script.

# .env
É um arquivo onde você coloca senhas, chaves de API, dados do banco e outras configurações que mudam quando o sistema sai do seu PC e vai para a hospedagem.
